Python plotly ktphanesi ile interaktif grafik hazrlama

plotly interaktif grafikler hazrlayabileceiniz bir python paketidir. Resmi sayfalarnda kendilerini "Plotly provides a web-service for hosting graphs!" eklinde tanmlyorlar. Yani hazrladnz grafikleri plotly sunucularnda insanlara aabilirsiniz.

Tabi iin bir de "open source" boyutu var. Biz bu yazmzda plotly paketinin "open source" ksmn kullanarak interaktif grafikleri nasl yapabileceimizi inceleyeceiz.

Ref: https://plot.ly/python/

trace = go.Scatter(
    x = random_x,
    y = random_y,
    mode = 'markers',
    
    # dict formatnda bir marker veri yaps ekliyoruz
    marker = dict(
        
        # elmas eklinde olsun
        symbol = "diamond",
        
        # Biraz saydamlk verelim
        opacity = 0.5,
        
        # Boyutunu ayarlayalm
        size = 10,
        
        # rengi hex olarak verebiliyoruz. 
        # Baka formatlar da var tabi ama hex en gzeli
        color = '#FFA07A', 
        
        # evresine siyah bir izgi ekelim
        # Bu izgi ksm da 
        line = dict(
            color = '#000000',
            width = 2
        )
    )
)

# iplot kullanarak izdiriyoruz
iplot([trace])

matplotlib ile grafik hazrlayp plotly ile kullanmak

matplolib en sevdiim python ktphanelerinden biridir. Fakat bazen grafik zerinde veriyi daha interaktif bir ekilde inceleyebilsem dediim olmutur.

plotly bu "interaktiflik" ihtiyacn giderebilir belki. imdi matplotlib ile hazrladmz bir grafii plotly ile gsterebilir miyiz ona bakalm.

Bunun iin ok paral bir matplotlib figr oluturalm ncelikle

In[25]:
import matplotlib.pyplot as plt
# Burada figr ksm kritik bunu almamz lazm
fig, axes = plt.subplots(nrows=2, ncols=2,figsize=(10,10))

# biraz veri hazrlayalm
y = np.random.uniform(low=-5,high=5,size = 100)
x = np.random.uniform(low=-5,high=5,size = 100)

# axes iki boyutlu bir matris olarak dndrlyor
# sol st eksene plot izdirelim
axes[0][0].plot(x,y)
axes[0][0].set_title("plot")

# sa st eksene scatter izdirelim
axes[0][1].scatter(x,y,c='r')
axes[0][1].set_title("scatter")

# sol alt eksene x histogram, sa alt eksene y histogram
axes[1][0].hist(x,color="g",edgecolor="yellow")
axes[1][0].set_title("x histogram")
axes[1][1].hist(y,color="m",edgecolor="k")
axes[1][1].set_title("y histogram")

plotlydeki egzotik grafik rnekleri

sunburst

imdi sizlere ok k bir grafik gsteceim, hem de animasyonlu :) Yani grafiin zerine tkladnzda alt dalmlar grmeniz mmkn. Grafik temelde bir piechart gibi ama ok katmanl.

Bir rnek zerinden gidelim diyelim ki "Gelir" ve "Gider" olarak iki kaleminiz var ve bu kalemlerdeki harcamlar blerek gstermek istiyorsunuz.

ncelikle "labels" ksmna tm kalemleri, "parents" ksmna da bal olduklar ana kalemleri yazyorsunuz. Yani rneimizde bu kalemlerin hiyerarisi ve bunlarn deerleri aadaki gibi:

  • Bte (0)
    • Gelir (0)
      • Danmanlk (100)
      • Mhendislik (90)
      • Eitim (60)
    • Gider
      • Malzeme (100)
      • Ulam (20)
      • Personel (80)

Burada dikkat etmeniz gereken bir baka husus da eer alt kalemleri kullanarak bir st kalemi belirlemek istiyorsanz, st kalem iin "values" ksmna 0 girmeniz gerektii.

In[23]:
# grafik objelerini de go eklinde import edelim
import plotly.graph_objs as go
from plotly.offline import download_plotlyjs, init_notebook_mode, plot, iplot

trace = go.Sunburst(
    labels=["Bte", "Gelir", "Gider", "Malzeme", "Ulam", "Personel", "Danmanlk", "Mhendislik", "Eitim"],
    parents=["", "Bte", "Bte", "Gider", "Gider", "Gider", "Gelir", "Gelir", "Gelir" ],
    values=[0, 0, 0, 100, 20, 80, 100, 90, 60],
    outsidetextfont = {"size": 20, "color": "#377eb8"},
    marker = {"line": {"width": 2}},
)

layout = go.Layout(
    margin = go.layout.Margin(t=0, l=0, r=0, b=0)
)

iplot(go.Figure([trace], layout), filename='basic_sunburst_chart')

sankey

Bir ak betimlemek veya birbiri ile orantlarn gstermek iin sankey diyagram kullanlr. Yaratc kullanm alanlar da vardr bu grafiin. rnein bir seimde partiler arasnda gei yapan semenleri analiz etmek iin kullanlabilir. Ref: cilekagaci.com

sunburst diyagramndaki rnei uygulayalm isterseniz. Burada source, target ve value deerlerini dikkatli ayarlamamz gerkiyor.

In[24]:
import plotly.plotly as py

data = dict(
    type='sankey',
    node = dict(
      pad = 15,
      thickness = 20,
      line = dict(
        color = "black",
        width = 0.5
      ),
      label=["Bte", "Gelir", "Gider", "Malzeme", "Ulam", "Personel", "Danmanlk", "Mhendislik", "Eitim"],
      color = ["yellow", "blue", "red", "red", "red", "red","blue","blue","blue"]
    ),
    link = dict(
      source = [0,0,1,1,1,2,2,2],
      target = [1,2,6,7,8,3,4,5],
      value = [200,250,100,20,80, 100, 90, 60]
  ))




fig = dict(data=[data])
iplot(fig, validate=False)
